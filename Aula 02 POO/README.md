# Segunda Aula de POO

## Assuntos abordados na aula
1. Criar um repositório no github:
    - Fomos na opção de new repository e criamos um novo colocando somente o título e descrição.
1. Criar um .gitignore
    - Utilizamos o site: https://www.toptal.com/developers/gitignore/
    - No site foi adicionado as tags: linux; windows; intellij+all; VisualStudioCode; java; gradle.
1. Criar uma LICENSE
    - Para criarmos uma licensa utilizamos o site: https://choosealicense.com/
    - No site foi escolhida a opção "MIT license"
    - Para colocar a Licença no Github, copiamos o que estava no "MIT license", criamos um arquivo de nome "LICENSE" no git e colamos a escrita do site no arquivo do github
1. Adicionar um arquivo no github e depois tentar dar commit de outro arquivo por fora, a fim de forçar um git merge
    - Para essa tarefa criamos um arquivo utilizando o codespace e depois tentamos enviar um arquivo que criamos só no computador para o git através do git push. Como o destino estava diferente da main foi necessário fazer um git pull e depois um git push para conseguir corrigir o erro e deixar os dois arquivos no repositório.
